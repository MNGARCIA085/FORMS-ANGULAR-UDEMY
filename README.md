# FORMS-ANGULAR-UDEMY

## Description

Template and reactive forms


## How to use the app

- Clone the repo
- Server

Install json-server

...$ cd server

..server$ json-server --watch db.json


- Client

Install dependencies and start the client.

...$ cd client

...client$ npm install

....client$ ng serve

<hr>

Go to localhost:4200
